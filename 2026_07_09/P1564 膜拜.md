## P1564 膜拜
## 题意
$$\begin{aligned}
 &n个人,每个人有一个编号i,(i\in{1,2}),选择连续片段分配房间,,要求满足一下两个条件之一:\\&1.单个教室人员编号一致 \\&2.不同编号的人员差值不超过m\\&最少要多少件房屋才能放 n个人? \space 1≤n≤2500，0≤m≤2500
\end{aligned}$$

## 思路
$$\begin{aligned}
      &f[i]=前i个人所需要的最少房间数\\&f[i]=min(f[j]+1),这里需要满足,0\leq j<i,以及[j+1,i]是符合条件的\\&也就是说我们需要统计这个区间的人员偏好数量,可以使用前缀和维护\\&f[0]=0
\end{aligned}$$
## 易错点
为了设计该状态,我们需要思考最后一个房间装的是哪一段同学? 

注意范围!!

## 代码/实现
```c++
const int inf = 999999999;
void solved(){
    int n,m;
    cin>>n>>m;
    vector<int> a(n+1,0),sum1(n+1,0),sum2(n+1,0),f(n+1,inf);
    for(int i=1;i<=n;i++){
        cin>>a[i];
        sum1[i]=sum1[i-1]+(a[i]==1);
        sum2[i]=sum2[i-1]+(a[i]==2);
    }
    f[0]=0;
    for(int i=1;i<=n;i++){
        for(int j=0;j<i;j++){
            int cnt1 = sum1[i]-sum1[j];
            int cnt2 = sum2[i]-sum2[j];
            if(cnt1==0||cnt2==0||abs(cnt1-cnt2)<=m){
                f[i]=min(f[i],f[j]+1);
            }
        }
    }
    cout<<f[n];
}
```
## 一句话复盘
>本质就是:前 `i` 个同学的最优划分，一定可以看成：前 `j` 个同学的最优划分 + 最后一个合法区间 `[j+1,i]`。

这类题最常见的思考方式就是：
> 枚举最后一段从哪里开始。
