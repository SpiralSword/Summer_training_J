## P1103 书本整理
## 题意
$$
\begin{aligned}
    &给定n本书,高乘宽为h*w,先按高度升序排列.\\&定义杂乱度:相邻书的宽度差的绝对值之和,从中抽取k本书使得杂乱度最小\\&（1<n≤100,1≤k<n）保证每本书高度不同 ,1<h,w<200
\end{aligned}
$$
## 思路
$$
\begin{aligned}
     &将问题进行转换,拿走k本书,看为选n-k本书\\
     &f[i][j]=考虑前i本书,保留j本且第j本一定是第i本书\\
     &f[i][j]=min(f[i][j],f[p][j-1]+abs(w[i]-w[p])),p<i,j,\leq min(n-k,i)\\
     &f[i][1]=0
\end{aligned}
$$
## 易错点
初始化为最大值
## 代码/实现
```c++
void solved(){
    int n,k;
    cin>>n>>k;
    vector<pii> a(n+1,{0,0});
    for(int i=1;i<=n;i++) cin>>a[i].first>>a[i].second;
    sort(all(a));
    int ans = 99999999;
    vector<vector<int>> f(n+1,vector<int>(n+1,999999999999)); // f[i][j] 第i本书,保留j本一定保留第i本的最小差异度
    for(int i=1;i<=n;i++) f[i][1]=0; // f[i][j] = min(f[p][j-1]+abs(w[i]-w[p]))
    for(int i=1;i<=n;i++){
        for(int j=2;j<=min(n-k,i);j++){
            for(int p=1;p<i;p++){
                f[i][j]=min(f[i][j],f[p][j-1]+abs(a[i].second-a[p].second));
            }
        }
    }  
    for(int i=n-k;i<=n;i++) ans = min(ans,f[i][n-k]);
    cout<<ans;
}
```
## 一句话复盘
删除 `k` 本书，等价于保留 `n-k` 本书。  
排序后，在原序列中选一个子序列，使相邻宽度差最小。  
所以用 DP 枚举“当前保留哪本书，上一本文保留哪本书”。
